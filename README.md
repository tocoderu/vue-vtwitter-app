## Build Setup:

```bash
# Download repository:
git clone https://github.com/vedees/tocode-youtube-vtwitter vtwitter

# Go to the app:
cd vtwitter

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8081/
npm run serve

# Output will be at dist/ folder
npm run build
```
